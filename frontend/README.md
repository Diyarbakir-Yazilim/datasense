# Frontend

This directory contains the user interface for the DataSense application.

It is built using React and Vite. The frontend provides an interactive, drag-and-drop interface for users to upload their datasets and view the intelligent cleaning decisions and data visualizations generated by the AI engine.

## Getting Started

To run the frontend locally:

1. Ensure you have Node.js installed.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
